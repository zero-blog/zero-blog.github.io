Hexo新建文章，拟定标题并分类

关于Hexo博客 Hexo 2019/06/01

1. 创建文件.

  • 方式一：
    hexo new “hexo教程”

  • 方式二：

    在博客目录下的 source 之中的 _posts 的文件夹中创建 *.md

2. 新建文章，拟定标题并分类.

  1. — 这个符号是前后都必须有，而且是三个一起哦，之后就可以写文章了内容了.

4. 执行 hexo 命令.

  • hexo clean 清空资源文件，可选操作
  • hexo g 重新生成资源文件
  • hexo server 本地发布文章，默认端口4000
  • hexo s -p 5000 或者给定端口5000，端口可改
  • 本地发布文章后，查看效果：http://localhost:4000
  • 如果端口改了，查看效果时，端口也要改
  • hexo deploy 发布文章到 github
  • 查看个人博客效果
